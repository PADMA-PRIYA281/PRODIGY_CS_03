# Password Complexity Checker

## Description

This project is a Password Complexity Checker developed using Python. It evaluates the strength of a password based on various security criteria and provides feedback to help users create stronger passwords.

## Features

* Checks password length.
* Verifies the presence of uppercase letters.
* Verifies the presence of lowercase letters.
* Checks for numbers.
* Checks for special characters.
* Provides password strength feedback (Weak, Moderate, Strong).

## Technologies Used

* Python

## How to Run

1. Open the terminal.
2. Run:
   python password_checker.py
3. Enter a password to check its strength.

## Password Strength Criteria

* Weak: Meets 0–2 requirements.
* Moderate: Meets 3–4 requirements.
* Strong: Meets all 5 requirements.
